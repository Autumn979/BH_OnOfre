O que é?

> BH OnOfre Single
-> ESP01
-> 2 GPIO's 
-> 1 relé solid state permite ligar e desligar sem ruido mecânico
-> RX TX
  
> BH OnOfre Dual
-> ESP12
-> 9 GPIO's 
-> 2 relé solid state permite ligar e desligar sem ruido mecânico 
-> RX TX
-> ADC
